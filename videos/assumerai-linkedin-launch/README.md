# Assumerai LinkedIn Launch Videos

Campaign scaffold for the Assumerai LinkedIn launch video project.

## Structure

- Shared design system: `shared/assumerai-launch.css`
- Shared motion helpers: `shared/assumerai-motion.js`
- Candidate composition: `candidate/index.html`
- Hiring teams composition: `hiring-teams/index.html`
- Source assets: `assets/`
- Render output: `renders/`

## Font Guidance

Use `font-family: "Geist", Arial, sans-serif` for primary type and `font-family: "Instrument Serif", Georgia, serif` for accent phrases. Do not add web font imports; HyperFrames embeds supported fonts when these font-family names are written in the composition CSS, with the listed fallbacks available if needed.
