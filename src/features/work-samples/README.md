# Work Samples

This feature owns short, role-specific work samples for the interview module.
The MVP catalog covers:

- `coding`: a Python API-helper bugfix with deterministic tests.
- `writing`: an SDR follow-up and CRM note.
- `analysis`: a tech-risk control-gap assessment.

Rubrics are versioned separately from work-sample definitions and scoring
formula versions. Reviewers must grade submitted work product and evidence
only. The catalog uses a central prohibited-signal policy for direct age,
protected attributes, accent, personality, biometric, face, emotion, health,
family-status, and nationality signals.

Safe execution is intentionally limited to local/static review or local static
tests. This task does not add a production sandbox, external-service access, or
automated employment decisions. Every work-sample result remains a
recommendation that requires meaningful human review.
